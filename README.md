本レポジトリは長野県立大学伊藤享平の卒業論文管理のために作成されました。


## Install

```
  git clone https://github.com/kyoheiii/fanaticismD
  cd fanaticismD
  docker-compose up -d --build
```

## Usage
Docker Desktopの場合
http://localhost:8787

RStudio Server にアクセス後、
https://github.com/kyoheiii/fanaticism 
をクローンしてください
